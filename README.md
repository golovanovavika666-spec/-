# -2D шутер про зомби с видом сверху, где игроку предстоит выжить в условиях постоянного давления со стороны нежити. Волны становятся сложнее, врагов больше, а ресурсов всё меньше — приходится выбирать, как сражаться и когда отступать.

В проекте реализованы:
• динамичные волны зомби
• система урона и смерти
• стрельба и боезапас
• интерфейс игрока
• визуальные эффекты
